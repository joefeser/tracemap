# Legacy Evidence Packs

This directory is the approved tracked promotion root for public-safe legacy
evidence packs generated by `tracemap evidence-pack promote`.

Evidence packs are redacted summaries over TraceMap artifacts. They preserve
rule IDs, evidence tiers, coverage labels, source labels, safe provenance,
gaps, and limitations. They do not copy raw scan artifacts, local paths, raw
SQL, config values, source snippets, remotes, or private sample names.

Generated local output belongs under `.tmp/legacy-evidence-packs/`.
