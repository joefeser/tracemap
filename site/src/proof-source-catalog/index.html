Proof source catalog

Map each public route back to the source material that bounds its wording.

The catalog is a public-safe orientation layer for managers, reviewers, and claim-checking bots. It links route wording to proof paths, source artifact families, rule IDs or rule families, evidence tiers, coverage labels, limitations, and non-claims.

Public claim level: demo. Page-level demo status describes this new catalog surface; it does not downgrade shipped rows or upgrade concept or hidden rows. No public conclusion without evidence.

Non-claims

The catalog does not strengthen the source material it indexes.

  • It does not prove runtime behavior, production traffic, endpoint performance, outage cause, release safety, operational safety, release approval, AI impact analysis, LLM analysis, or complete product coverage.
  • It does not publish raw facts, raw SQLite, analyzer logs, raw source snippets, raw SQL, config values, secrets, local absolute paths, raw remotes, generated scan directories, or private sample names.
  • Concept rows are not shipped capabilities, demo rows are scoped to public-demo proof, and hidden rows should not be repeated as public claims.
