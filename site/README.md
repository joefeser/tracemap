# TraceMap Tools Site

This static site publishes public TraceMap product and documentation content for
`tracemap.tools`.

Amplify builds from the repository root with `amplify.yml`, treats `site/` as the
app root, runs `npm run build`, and publishes `site/dist`.

## Deployment

Use the existing `joefeser/tracemap` repository.

```text
Branch: main
App root: site
Prebuild: npm ci
Build command: npm run build
Output directory: dist
Node: 20+
Domain: tracemap.tools
```

Validation URLs:

- `https://tracemap.tools/`
- `https://tracemap.tools/robots.txt`
- `https://tracemap.tools/sitemap.xml`
- `https://tracemap.tools/evidence/`
- `https://tracemap.tools/demo/`

## Local Development

```bash
npm install
npm run dev
npm run build
```

`src/` is the editable source. `dist/` is generated by the build and is not the
source of truth.

## Site Work Conventions

- Start future site specs with `site-`, for example
  `.kiro/specs/site-tracemap-tools-discovery/`.
- Use a separate worktree for site slices.
- Do not edit `site/dist/` by hand.
- For layout changes, run `npm run build` and do a browser sanity check at
  desktop and mobile widths.
