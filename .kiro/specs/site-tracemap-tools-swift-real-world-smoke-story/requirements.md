# Site: Swift real-world smoke story

Status: implemented
Public claim level: shipped

## Intent

Publish the now-landed Swift v0 real-world API-client smoke validation as a public-safe site story.

## Requirements

- Add a public route for Swift real-world smoke evidence.
- Make Swift v0 easier to find from public entry points.
- Anchor the route to the checked-in validation guide and smoke script.
- Include pinned public sample labels, repository names, pinned SHAs, and why each sample exists.
- State only static evidence discovery and artifact-generation claims.
- Avoid runtime/API correctness, simulator/device, Xcode build, SwiftPM restore, app execution, endpoint reachability, production telemetry, package compatibility, complete Swift semantic analysis, or impact claims.
- Keep raw generated artifacts, local paths, clone URLs, raw remotes, source snippets, raw SQL, credentials, hostnames, config values, and runtime observations out of public copy.
