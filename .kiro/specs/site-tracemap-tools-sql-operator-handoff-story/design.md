# Site SQL Operator Handoff Story Design

The route uses the existing TraceMap public-site system and typography. A
manager-first question leads into an allowlisted synthetic packet illustration:

- ordered categorical context checkpoints;
- static prerequisite statuses;
- protected handling categories;
- intended milestones versus validation-step presence;
- stop conditions, gaps, owner questions, and non-claims.

The illustration is descriptive HTML, not copied scan output or executable SQL.
Every example row retains a public rule ID, evidence tier, coverage label,
repo-relative synthetic span, and extractor version. The manager packet links
to the route; broader proof and limitation pages provide inbound navigation.
